How to get Google OAuth credentials
  1. Go to console.cloud.google.com and sign in.
  2. Create a new project (or select an existing one).
  3. In the left menu go to APIs & Services → Library.
    Search for Photos Library API and click Enable.
  4. Go to APIs & Services → OAuth consent screen.
    Choose External, fill in App name (e.g. "Landscape Planner"), your email, then Save and Continue through all steps.
  5. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID.
    Application type: Web application.
    Under Authorized redirect URIs add:
    {{ request.base_url }}auth/google/callback
  6. Click Create. Copy the Client ID and Client Secret shown.
  7. Paste them in the form below and click Save.
